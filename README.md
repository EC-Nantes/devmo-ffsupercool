Projet FFSUper Cool

Ce projet a pour but de créer une application qui permet aux étudiants d'obtenir toutes les informations qui leur sont nécessaires pour le sport universitaire.
Il se base sur un site web déjà existant mais pas très pratique

Nous avons convenu d'implémenter un écran avec un UiState
Nous avons donc choisi d'implémenter l'écran d'accueil avec des équipes suivies déjà insérées, et il serait possible de supprimer ces équipes.
L'ajout de nouvelles équipes suivies est sensé se faire via un onglet paramètres avec de nombreux filtres, que nous n'allons pas implémenter
Et le reste de la page (la liste des matchs) est sensé correspondre aux équipes suivies également mais nous n'allons pas l'implémenter
